# LaTeX Thesis Class

A scalable LaTeX class for university theses, designed to support multiple thesis levels (Bachelor, Master, PhD) and easy customization (e.g., university template, language, layout).

## Features
- Thesis level support: Bachelor / Master / PhD
- Intended to support different university formatting models (templates)
- Designed to be adaptable to different languages
- Clean and reusable class structure
